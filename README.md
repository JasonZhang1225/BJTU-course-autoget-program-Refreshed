# 北京交通大学自动抢课工具

一个基于 Electron 的北京交通大学自动抢课工具,支持验证码自动识别。

## 功能特点

- 🚀 自动登录教务系统
- 🔄 自动刷新课程信息
- 🎯 支持多课程同时抢课
- 🔍 智能识别验证码（登录页算式 OCR + 选课页 Qwen 识别/手动填写）
- 🔒 本地保存配置信息
- 📊 实时显示抢课状态

## 使用说明

1. 下载并安装程序
2. 启动程序后输入以下信息:
   - 学号
   - 密码
   - 阿里云千问 Dashscope API Key (用于选课验证码识别)
   - 待选课程名称 (多个课程用英文逗号分隔)
3. 点击“启动服务"然后点击"开始抢课"按钮即可开始自动抢课

## 注意事项

- 请确保输入的课程名称准确
- 建议使用稳定的网络连接
- 请勿频繁开关程序
- 使用过程中如遇问题,可点击"停止抢课"

## macOS 用户须知

由于未进行 Apple 开发者签名，macOS 用户首次打开应用会提示"无法验证开发者"或者“已损坏”。请将应用拖入 `/Applications` 后，在终端运行以下命令：

```bash
sudo xattr -rd com.apple.quarantine /Applications/北京交通大学自动选课软件.app
```

之后即可正常打开。

## 技术栈

- Electron
- Vue.js 
- Python
- WebSocket
- DdddOcr（自建 ONNX 运行时，登录算术验证码）
- 通义千问 Qwen VL OCR（选课中文验证码识别）

## 自定义修改

如果需要适配最新版本的教务系统,可以修改以下文件:

```
%appdata%/../Programs/web-get-course/resources/app.asar.unpacked/src/login.py
```

该文件包含了登录相关的核心逻辑,可以根据教务系统的更新进行相应调整。
